This github repository (https://github.com/rjdossan/METU_UMAP) contains matlab functions to help visualise multi-platform metabolomics datasets using UMAP, to create and evaluate robust intra-dataset clusters, and to evaluate inter-dataset cluster reproducibility. 

Some help functions from the github repository (https://github.com/rjdossan/HELP_RPtools) are also required, as well as the packages below. 

The M2S package (Pinto et al, 2022) code for matching features across two datasets is required for some of the functions above, and for the inter-dataset methodology, and it is available at (https://github.com/rjdossan/M2S) 

We used an implementation of UMAP implementation (accessed in 2022) by Connor Meehan, Jonathan Ebrahimian, Wayne Moore, and Stephen Meehan, obtained from the MATLAB Central File Exchange (Uniform Manifold Approximation and Projection, UMAP): https://www.mathworks.com/matlabcentral/fileexchange/71902.

For the robust intra-dataset cluster methodology, we used functions for the definition of Louvain communities from the github repository CanlabCore, by the Cognitive and Affective Neuroscience Lab (Tor Wager, PI) (accessed in 2026): https://github.com/canlab/canlabcore.
